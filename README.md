# Django로 Instagram만들기

[인스타그램 만들기 프로젝트](https://lhy.kr/lecture/django/instagram/)의 소스코드 저장소

## Requirements

- Python > 3.6
- pip packages 
	- `pip install -r requirements.txt`

## Run

```shell
git clone <저장소> django-instagram
cd django-instagram
cd instagram
python manage.py runserver
```

```localhost:8000```확인
